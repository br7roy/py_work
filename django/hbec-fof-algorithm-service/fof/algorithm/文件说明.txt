资产配置-均值方差模型为大类资产配置模块的封装好的一个类

输入数据均为json格式
输出为资产配置模型的配置结果


template_fundFactorEvaluation为基金单指标回测类
输入值均为字符串（可进一步约定为json格式）
输出为基金指标评价结果及净值曲线图

运行该类，需要调用数据库数据，本例中我们采用了华宝金工自建的mongodb数据库作为案例

运行该类，需要加载相关函数，如数据库数据调取函数，基金指标函数等，均已写入cusFundfunction中
运行该程序，由于涉及与云端数据库交互，运行较慢，若调取本地数据库，经测试10秒即可完成

程序运行环境：
Python 3.6.5 
需要额外按照 pymongo包，pip install 即可
